# Worker Effort Profile: medium

Use this profile for straightforward or moderately scoped implementation work:

- small feature additions
- localized bug fixes
- tests and documentation
- narrow refactors
- low-ambiguity tasks with a clear plan

Guidance:

- Prefer simple, targeted changes.
- Avoid broad redesigns unless explicitly requested.
- Validate with the smallest relevant test/typecheck/lint command when practical.
- If the task is substantially more complex than this profile is appropriate for, do the safest bounded subset and note whether `effort=high` or `effort=xhigh` would be more suitable.
